📄 Resume Analyzer & ATS Score Predictor

An AI-powered web app that analyzes resumes against job descriptions and generates an ATS score using NLP techniques. It highlights matched/missing keywords and provides smart suggestions to improve resume compatibility.

🚀 Features
Upload resume (PDF)
Paste job description
ATS score prediction (0–100)
Keyword matching (matched & missing)
Skill categorization (technical & soft skills)
Resume improvement suggestions
🛠️ Tech Stack
Python
Streamlit
NLTK & spaCy
Scikit-learn (TF-IDF, Cosine Similarity)
pdfplumber
